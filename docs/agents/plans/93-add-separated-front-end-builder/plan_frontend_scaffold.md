# Front-End Scaffold Plan

Step 1 of [plan.md](plan.md). Create the `frontend/` directory with all configuration files and the placeholder entry point.

Reference: `../weave/frontend/`.

---

## Directory Structure

Following the component pattern from `../navi/frontend/src/components/` — each non-trivial component is split into three layers (component, controller, helper). See `docs/agents/frontend.md` for the full architecture reference.

```
frontend/
  assets/
    css/
      styles.css              # custom CSS
      main.scss               # custom SCSS
    js/
      components/
        elements/             # reusable UI building blocks
          controllers/        # logic for elements
          helpers/            # JSX factories for elements
        pages/                # top-level route components
          controllers/        # logic for pages
          helpers/            # JSX factories for pages
        App.jsx               # root component (placeholder for now)
      main.jsx                # entry point
  spec/
    example_spec.js           # smoke test
  index.html
  package.json
  vite.config.js
  eslint.config.mjs
  yarn.lock                   # generated by yarn install
```

## `assets/js/main.jsx`

Mirror `../weave/frontend/assets/js/main.jsx`:

```jsx
import { createRoot } from 'react-dom/client';
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
import App from './components/App';

import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';

import '../css/styles.css';
import '../css/main.scss';

const queryClient = new QueryClient();

(function () {
  createRoot(document.getElementById('root')).render(
    <QueryClientProvider client={queryClient}>
      <App />
    </QueryClientProvider>
  );
})();
```
